# ⚡ БЫСТРЫЙ СТАРТ

> **Запустите одну команду после добавления файла — все остальное сделается автоматически!**

---

## 🚀 ОСНОВНАЯ КОМАНДА

После добавления **любого** файла просто запустите:

```bash
make process
```

**Это все!** Система автоматически:
1. ✅ Переместит файлы из Inbox в правильные папки
2. ✅ Добавит YAML метаданные (type, created, updated, tags)
3. ✅ Исправит теги (приведет к единой системе `#tech/`, `#pattern/`, etc.)
4. ✅ Проверит и исправит ссылки
5. ✅ Обновит MOCs

---

## 📋 ЧАСТЫЕ СЦЕНАРИИ

### Добавили файл в Inbox

```bash
# 1. Создали файл
echo "# Новый паттерн" > 00-Inbox/Новый\ паттерн.md

# 2. Запустили обработку
make process
```

### Проверить систему

```bash
make check
```

### Только переместить из Inbox

```bash
make move-from-inbox
```

### Только исправить теги

```bash
make fix-tags
```
