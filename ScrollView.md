#frontend #reactnative 

`ScrollView` - общий контейнер для прокрутки, который может содержать несколько компонентов и представлений. Прокручиваемые элементы могут быть разнородными.
Он лучше всего подходит для представления небольшого кол-ва объектов ограниченного размера. Все элементы и представления отображаются, даже если их нет на экране. 
Для больших списков этот компонент не рекомендуется.

Свойства:
- `pagingEnabled` - позволяет листать представления с помощью жестов пролистывания
- `maximumZoomScale && minimumZoomScale` - можно использовать на iOS вместе с одним элементом для зума этого элемента
