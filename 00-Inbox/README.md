# 📥 INBOX (Входящие)

> **Временная папка для быстрых заметок**

---

## 🎯 НАЗНАЧЕНИЕ

Эта папка предназначена для **быстрого захвата информации** без размышлений о том, куда её поместить.

**Правило:** Обрабатывайте заметки из Inbox **в течение 24 часов**!

---

## ✅ КАК РАБОТАТЬ С INBOX

### Когда создавать заметку в Inbox:
- ✏️ Быстрая идея или мысль
- 📝 Заметка с встречи
- 🔗 Полезная ссылка
- 📚 Информация из статьи/видео
- 💡 Внезапный инсайт

### Когда ОБРАБАТЫВАТЬ Inbox:

#### Каждое утро (5 минут):
1. Просмотрите новые заметки в Inbox
2. Быстро решите: важна ли заметка?
   - **Да** → переместите в нужную папку (`01-Projects/`, `02-Areas/`, `03-Resources/`)
   - **Нет** → удалите

#### Каждый вечер (10 минут):
1. Обработайте все заметки за день
2. Добавьте метаданные (теги, свойства)
3. Создайте связи с существующими заметками через `[[ссылки]]`
4. Обновите соответствующие MOCs

---

## 📋 ЧЕКЛИСТ ОБРАБОТКИ ЗАМЕТКИ

Для каждой заметки в Inbox:

- [ ] **Классифицировать:**
  - [ ] Это проект? → `01-Projects/`
  - [ ] Это обучение? → `02-Areas/Learning/`
  - [ ] Это справочник? → `03-Resources/`
  - [ ] Это личное? → `02-Areas/Personal/` или `02-Areas/Health/`

- [ ] **Добавить метаданные:**
  - [ ] Указать `type` (note, project, meeting, etc.)
  - [ ] Добавить теги (максимум 3-5)
  - [ ] Указать `status` (active, archived, draft)

- [ ] **Создать связи:**
  - [ ] Добавить `[[ссылки]]` на связанные заметки
  - [ ] Обновить соответствующие MOCs
  - [ ] Добавить обратные ссылки в связанных заметках

- [ ] **Переименовать:**
  - [ ] Использовать формат `YYYY-MM-DD - Название.md` (для временных)
  - [ ] Или просто `Название.md` (для перманентных знаний)

---

## 🚫 ЧТО НЕ ДЕЛАТЬ

❌ Оставлять заметки в Inbox более 24 часов  
❌ Создавать подпапки внутри Inbox  
❌ Организовывать заметки в Inbox — это не для организации!  
❌ Накапливать заметки — регулярно очищайте Inbox  

---

## 💡 СОВЕТЫ

1. **Делайте заметки быстро** — не думайте о структуре, просто пишите
2. **Обрабатывайте регулярно** — лучше по 5-10 минут ежедневно, чем 2 часа раз в неделю
3. **Будьте безжалостны** — если заметка не важна через 24 часа, удалите её
4. **Создавайте связи** — каждая заметка должна быть связана хотя бы с одной существующей
