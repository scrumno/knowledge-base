## 🚀 ЧТО ДЕЛАТЬ ДАЛЬШЕ

### 1. Начните использовать систему

#### Утро (5 минут):
1. Откройте `99-MOCs/Home.md` — главная навигация
2. Проверьте `00-Inbox/` — обработайте новые заметки
3. Просмотрите `01-Projects/` — задачи на день

#### Для создания заметок:
- Быстрая заметка → создайте в `00-Inbox/`
- Новый проект → используйте `Templates/Project-Template.md`
- Встреча → используйте `Templates/Meeting-Template.md`

### 2. Обработайте Inbox

Файлы в `00-Inbox/` нужно обработать:
- Просмотрите каждый файл
- Определите категорию (проект/обучение/ресурс)
- Переместите в нужную папку
- Добавьте метаданные (опционально)
- Создайте связи `[[ссылки]]`

### 3. Добавьте метаданные (опционально)

Можно добавить YAML-метаданные в начало файлов для лучшей организации.  
См. `README_SYSTEM.md` → раздел "YAML-метаданные (Properties)".

### 4. Обновите MOCs

После обработки заметок обновите соответствующие MOCs:
- `Projects-MOC.md` — для проектов
- `Learning-MOC.md` — для обучения
- `Patterns-MOC.md` — для паттернов
- `Bitrix-MOC.md` — для Bitrix

1. **Обработайте Inbox ежедневно** — не оставляйте заметки в Inbox более 24 часов
