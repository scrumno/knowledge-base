## 🚀 КАК ИСПОЛЬЗОВАТЬ

### После добавления файла:

```bash
make process
```

**Это все!** Система автоматически:
1. Переместит файл из Inbox в нужную папку
2. Добавит YAML метаданные
3. Исправит теги
4. Проверит ссылки
5. Обновит MOCs

---

### Проверка системы:

```bash
make check
```

---

### Просмотр всех команд:

```bash
make help
```
